Git stash is a way to store the changes you made on a branch without having to commit them or ad them too your project tht is being worked on.

Its perfect for when youre in the middle of something and you want to switch to doing something else.

Hint: if you see the following output when changing branches, youmight want to consieer using git stash. "please commit commit your changes or stash them before you switch branches."

To use Stash, simply run..

Git stash

You can view your stashed changes by runing git stash list This will show you a list of changes you have stored in the stash

You can apply any given stash to whichever part of your project you like by running the following command:

Git stash apply stash. i = the number that corresponds to the stash you want to apply.
